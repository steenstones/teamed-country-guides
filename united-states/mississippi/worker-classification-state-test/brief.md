## Editorial brief

### Target query
`mississippi worker classification test 2026`

### Core argument
Mississippi runs a dual-track classification system. The IRS 20-factor common-law test governs state income tax withholding and workers' compensation. MDES applies a separate ABC test under Miss. Code Ann. § 71-5-11(I)(15) for unemployment insurance eligibility. A contractor who passes the IRS test can still be an employee for UI purposes. Most employers only run one analysis.

### Audience
US-market employers considering remote or on-site hires in Mississippi; multi-state employers running regional contractor programmes across the south-east; HR and legal teams who need to understand the MDES ABC prong exposure.

### Key facts (all provisional — verify against source URLs)
- **Primary test:** IRS 20-factor common-law right-to-control test (DOR, workers' comp)
- **UI test:** ABC test under Miss. Code Ann. § 71-5-11(I)(15) — all three prongs required
- **MDES ABC Prong A:** freedom from control in fact and under contract
- **MDES ABC Prong B:** work outside usual course of hiring entity's business
- **MDES ABC Prong C:** worker customarily engaged in independently established trade or business
- **Civil penalty:** $50–$2,000 per worker per period of non-compliance (Miss. Code § 71-5-19)
- **No state-specific construction test**
- **No PAGA equivalent** — enforcement is regulatory, not private litigation

### Differentiation from California
California applies the ABC test across all purposes with PAGA exposure. Mississippi applies it only for UI. Lower stakes, but the dual-track architecture creates an identical Prong B failure point for the same set of roles.

### Differentiation from Texas/Florida
Texas and Florida run the IRS 20-factor test for all state purposes. Mississippi adds the MDES ABC layer for UI. Multi-state employers who route a single federal analysis cannot assume coverage in Mississippi.
